Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents that represents the structure of a webpage as a tree of objects. It allows scripts to update content, structure, and styles dynamically.

How do you select an element from the DOM?

To select an element from the DOM, you can use methods like: document.getElementById("id") document.querySelector("selector") document.getElementsByClassName("class") document.getElementsByTagName("tag")

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements. It's useful because it improves performance by reducing the number of event listeners and allows dynamic child elements to be handled automatically.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute() and getAttribute(), and modify styles using element.style.property. For example: element.setAttribute("class", "newClass") to change an attribute. element.style.color = "red" to update the style.
